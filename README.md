# Recursos interactivos de Matemáticas (3º ESO)

Colección de recursos interactivos en HTML diseñados para apoyar la 
enseñanza y el aprendizaje de las matemáticas en Educación Secundaria para 3º ESO.

## Contenido
- Actividades interactivas autoejecutables en el navegador
- Recursos visuales y manipulables para el bloque de geometría plana y espacial

## Cómo usar los recursos
Cada archivo HTML puede abrirse directamente en el navegador a través 
de su enlace correspondiente. No requiere instalación.

## Autoría
Material generado por Rubén García Erustes como parte del Trabajo Final de Máster / Máster Universitario en Formación del Profesorado / Generación y validación de recursos para el profesorado de matemáticas en Educación Secundaria mediante Inteligencia Artificial.
